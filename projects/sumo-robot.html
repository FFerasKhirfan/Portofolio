Autonomous Sumo Robot

A competitive robot designed to detect and push opponents out of a circular ring without human intervention.

Arduino C++ Sensors Strategic Thinking Robotics Electronics Motor Control

Project Overview

The Autonomous Sumo Robot is a competitive robotics project designed to compete in mini-sumo competitions. I designed and built a robot capable of detecting ring edges and opponent robots, implementing strategic movements, and executing offensive and defensive maneuversâ€”all without any human control during matches.

Key Features

  • Lightweight yet powerful design with high speed motors
  • 3D printed chassis optimized for weight distribution and stability
  • Two line sensors for detecting ring edges (white lines)
  • infrared distance sensors for opponent detection
  • Rapid acceleration and direction change capabilities
  • Specialized attack based on opponent behavior
  • Low-latency sensor processing for real-time decision making

Technical Implementation

The project involved several technical components working together:

1. Mechanical Design

The robot features a compact, low-profile chassis created with 3D printing. The design includes an angled front "plow" to get underneath opponents, a weighted rear section for enhanced stability, and a low center of gravity. High-torque DC motors with custom gear ratios were selected to provide the optimal balance of speed and pushing power. The wheels include high-friction rubber tires to maximize traction on the competition surface.

2. Electronics and Sensors

The robot uses an Arduino Nano as its main controller, connected to an array of sensors: two downward-facing line sensors for edge detection (white lines), and three IR proximity sensors for opponent detection. A dual H-bridge motor driver controls the motors, with power supplied by a high-discharge LiPo battery that provides bursts of current during rapid maneuvers.

3. Software Architecture

I developed a state-based control system in C++ that constantly evaluates sensor inputs and switches between different behavioral states: searching, tracking, attacking, and edge avoidance. The software employs sensor fusion techniques to improve detection reliability and includes a custom PID control system for precise motor movements. A key innovation was implementing a predictive tracking algorithm that anticipates opponent movement patterns.

4. Strategy Implementation

The robot employs multiple competitive strategies based on the specific situation. It begins with a 5-second scanning phase to locate the opponent, then executes either a direct charge, flanking maneuver, or feint-and-attack sequence. When near a ring edge, it rapidly calculates an optimal escape route while maintaining awareness of the opponent's position. The software also includes learning elements that adapt to opponent patterns throughout a match.
